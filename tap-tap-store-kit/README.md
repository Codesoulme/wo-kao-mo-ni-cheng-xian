## 1. 素材整理报告

### 1.1 应用图标 (`icons/`)

| 用途 | 文件 | 选片理由 | 修真用户吸引力 |
|---|---|---|---|
| 主图标·512·圆 | `icon_512x512_circle.png` | 主角图（07）正面照 | 玩家立刻能认出"这是主角"，与游戏"独一份"理念一致 |
| 主图标·512·方 | `icon_512x512_rounded.png` | 同上，角圆 18% | 适配方形位（部分商店列表） |
| 列表/小图 | 192/96/72/48 | 同视觉，多尺寸 | 商店列表+App 抽屉双适配 |

**修真色处理**
- 圆形版：紫色径向光晕 + 金色描边，仙气感 + 高级感
- 方角版：青色径向光晕 + 亮度 +12%，清爽不老气
- 修真核心配色：青 (75,235,230) / 紫 (138,79,255) / 金 (255,200,80) / 深空 (12,6,30)

**二次处理建议**
- 当前主图是主角图，剪影清晰但细节一般
- 设计师可在 PR/AE 里加：环绕金轮 + 太极八卦符号 + 修真门派徽记
- 命令骨架（设计师参考）：
  ```bash
  magick icon_512x512_circle.png -bordercolor "rgba(255,200,80,0.6)" \
    -border 8 -alpha set -channel A -evaluate multiply 0.9 +channel \
    icon_512x512_circle_premium.png
  ```

## 2. 修真品类自检要点

### 2.1 视觉自检
- 主色调：青/紫/黑 ✓
- 金色高光：题字/角标/描边 ✓
- 仙气光晕：图标径向 + 截图顶部渐变 ✓
- 修真符号：太极/道家/仙山/雷火/心魔 ✓（概念图原生提供）

### 2.2 文案自检
- 修真术语：渡劫/心魔/雷火/天象/道心/因果/灵根/仙缘 ✓
- AI 标注：含 AI 生成剧情 ✓
- 启动时长：首次启动约 30 秒 ✓
- 网络：需联网（AI 推演） ✓
- 修真 hook：八境遍历/因果轮回/独一份奇遇 ✓

### 2.3 修真用户最关心点
- AI 标注必须点明（修真用户对 AI 内容敏感） ✓
- 离线/网络需求必须说清（修真玩家不喜欢被"骗联网"） ✓
- 修真 hook 三大支柱（雷劫/心魔/天象）必须出现 ✓
- 修真术语要"地道"（渡劫/道心/因果/仙缘/灵根） ✓
- 一句话简介要"够修真"（含"独一份奇遇"修真感） ✓
