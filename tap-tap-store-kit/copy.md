## 3. 特色玩法（3-5 条 bullet · 修真用户偏好）

1. **AI 推演，每一次都是独一份奇遇**
   同一颗种子、不同开局，走出完全不同的仙路；天道不写死剧本。

2. **八境遍历，炼气到渡劫全程沉浸**
   炼气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘 → 渡劫 → 成仙，年推进、境界突破、灵根变异。

3. **雷火淬体 + 心魔试炼 + 天象异动**
   修真 hook 三大支柱：九重雷劫、三昧心火、紫气天象，每一项都由 AI 实时生成。

4. **因果轮回，抉择四宫格**
   善恶皆有回响，一念之差可能触发心魔也可能缔造仙缘。

5. **引擎校验，前端投影**
   数值与状态由 Engine 唯一权威，前端只承接槽位——你看到的每一行字、每一张卡，都经过校验。

## 6. TapTap 详情页"详细介绍"长版（备选·> 800 字）

### 缘起
市面上大多数修真游戏都是"数值堆砌 + 自动战斗"，玩家只需要无脑点"渡劫"。我们想做一个不一样的：
**修真不是无脑点渡劫，而是独一份的奇遇。**

### 我们怎么做的
1. **AI 是注册者，不是写作者**
   AI 不直接修改任何数值——它只"提议"，由 Engine 校验后才能生效。
   你不会看到"AI 一拍脑袋让主角成仙"——所有变更都经 ContentRegistry 注册、EffectResolver 结算、AI Boundary Validator 拦截。

2. **引擎是唯一权威**
   修真有修真自己的"天道"——天雷不可挡、灵根有限度、寿元有尽头。
   引擎强制这些规则，AI 越权即被打回。

3. **前端只承载与展示**
   状态面板、背包面板、战斗弹窗、Stream 浮现……所有 UI 槽位都按规则投影。

### 修真 hook·三大支柱
- **雷劫**——九重雷火，一念生死；渡劫成功开新境，失败入轮回。
- **心魔**——道心未坚即入魔；心魔试炼是修真最隐秘的关卡。
- **天象**——紫气东来、星河倒悬、灵潮涌动；天地异象给修真者最大机缘。

### 修真特色·细节
- **8 大境界** + **7 大突破** + **3 大心魔试炼** + **9 重雷劫** = 修真最全的体验
- **AI 实时推演** + **引擎硬约束** + **槽位可拓展** = 修真模拟器该有的样子
- **不需要肝**——年推进，玩家只做选择
- **不需要氪**——无内购，纯单机体验（仅 AI 调用需联网）

### 给修真玩家的话
- 如果你喜欢《凡人修仙传》的"凡人"质感，你会喜欢这里的"凡童起步"
- 如果你喜欢《诛仙》的"心魔"戏码，你会喜欢这里"道心试炼"
- 如果你厌倦了"自动战斗 + 数值碾压"，你会喜欢"选择 + 校验"的修真

**我靠模拟成仙，等你来渡劫。**
