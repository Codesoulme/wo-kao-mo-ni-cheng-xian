# 修仙世界常识（AI 提示词沉淀）

> 用途：**喂给 AI 的"世界常识"背景知识**，不是给玩家看的手册。  
> 形态：沉浸叙事中文，不暴露机制词（不写"境界 layer" "id" "配置" "cache" "ai"）。  
> 来源：综合《凡人修仙传》原作细节 + 跨 8 部热门修仙 / 仙侠作品的共识。  
> 接入：本文件通过 `src/lib/xianxia/prompt-zones.ts` 的 `SCENE_BEHAVIOR` 区 + `RETRIEVED_MEMORIES` 区按需注入；引擎按"灵根／境界／宗门／秘境" 等槽位在 prompt-zones 中钩子调用本文件。
>
> 与其它文档的关系：本文件不重复 `restrictions-detail.md` / `three-realms-detail.md` / `spirit-roots.md` / `tribulation-heart-demon.md` 等专题，本文沉淀**常识底色**（跨题材不变的部分），让 AI 在写剧情时"先有人味儿，再填细节"。

---

## 一、跨题材不变的世界底色

写修仙世界之前，先把以下潜规则吃透。这些是 8 部热门作品（《仙逆》《遮天》《完美世界》《凡人修仙之仙界篇》《剑来》《诛仙》《仙剑奇侠传》《雪中悍刀行》）去重之后留下的最稳共识：

### 1. 散修与宗门的真实处境

- 散修真实处境：资源被垄断、功法残缺、丹药昂贵、随时被高阶修士征用或抹杀。
- 宗门弟子：享受传承、洞府、灵脉、师长护短，代价是宗门任务、人情债、派系倾轧。
- 例：《仙逆》王林一路靠散修起步，吃尽残缺功法和资源断档的苦。

### 2. 前辈遗府与上古洞府

- 必有存在。它是"前人未竟之事 + 未用完的资源"的合集，**承担三重功能**：
  1. 功法传承；
  2. 剧情冲突焦点；
  3. 主角跳板的合理化（不能凭空给主角开挂，必须有一份"古人留下的"）。
- 例：《遮天》荒古圣地、《完美世界》下界八域皆处处是上古遗迹。

### 3. 天道、劫数、业力、功德（四词不是同义）

- **天道**：规则化身，可被感应、被触怒，可借不可逆。写"逆天而行"是危险语。
- **劫数**：强制关卡，最显化为雷劫、心魔劫；化神、证道、飞升皆必经。
- **业力**：与杀伐、屠戮、灭族挂钩，做多了反噬自身、招来心魔。
- **功德**：与"利众""守正""度化"绑定，常被用作减免天劫的筹码。
- 例：《仙剑奇侠传》蜀山弟子妄动杀念会被锁妖塔反噬；《剑来》陈平安的"善行"化作看不见的账。

### 4. 境界突破失败的代价（梯度很硬）

- 轻则重伤闭关 → 重则走火入魔 → 再重化形失败 / 道基崩碎 → 最重当场陨落或被天雷劈散真灵。
- **渡劫多次失败是常态**，不是个例。AI 写"一次就过"是不真。

### 5. 飞升 / 渡劫 / 红尘仙的结局分支

至少有四条线，且**飞升 ≠ 圆满结局**：

1. 成功飞升到上界（仙界/灵界）；
2. 渡劫失败转世重修；
3. 滞留下界，修成红尘仙 / 散仙；
4. 彻底陨落或被天雷劈散真灵。

## 二、力量尺度（不要降级、不要乱写）

AI 写打斗最容易把尺度写崩。这几条是"底线"：

| 操作 | 大致门槛 | 反例 |
|------|----------|------|
| 驾驭法器腾挪 | 炼气后期 | 炼气弟子"飞"是错的 |
| 御剑飞行（长距离） | 筑基后期 | 凡人"腾云"是错的 |
| 短距离瞬移 | 元婴以上 | 结丹期"瞬移"是错的 |
| 神识覆盖一域 | 化神以上 | 筑基期"神识覆盖百里"是错的 |
| 一念移山、袖里乾坤 | 化神以上 | 结丹期"移山"是错的 |
| 撕裂虚空 | 化神以上 | 元婴期"撕裂虚空"是错的 |
| 战场进入星空 / 星域 | 化神-大乘 | 把"星域战争"写在化神以下是错的 |

**力量刻度的铁律**：
- 境界压制几乎是铁律。低阶逆袭需要"底牌 + 环境 + 代价"三件套，且不可多写。
- 神识代替眼睛，是高阶战斗的预警核心（神识强弱直接决定战斗胜负）。
- "灵气无限"是错的。灵脉、灵石、灵眼是稀缺资源，常引发宗门争夺。

## 三、修仙社会与宗门的秩序逻辑

### 1. 宗门结构（修真界最稳定的组织形式）

| 层级 | 身份 | 实质 |
|------|------|------|
| 记名弟子 | 最外围 | 连核心功法都没资格看 |
| 外门弟子 | 杂务/采药/跑腿 | 没有独立洞府 |
| 内门弟子 | 被认定有培养价值 | 接触核心功法 |
| 真传 / 嫡传 | 大长老、掌门亲传 | 宗门的"未来" |
| 客卿 | 外聘的散修或其他宗门外援 | 给报酬做事 |
| 长老 | 普通 / 执事 / 太上长老 | 太上长老是宗门真正底蕴，平时闭关 |
| 掌门 | 明面管理者 | 权力大但未必是最强者（很多大宗门的实际掌权者是太上长老） |

资源分配铁律：资源按等级分配，功法严格保密，偷学功法是大忌。

### 2. 叛门代价

- 被认定叛逃后，原宗门派人追杀，且其他宗门不愿收留（怕得罪原宗门）。
- 例：《凡人修仙传》韩立离开天南时隐姓埋名了很久。

### 3. 修仙家族 vs 宗门 vs 散修

- 修仙家族是介于散修与宗门之间的存在：有自己的血脉传承、灵田、矿脉，但势力远不如宗门。
- 家族亲情有时比宗门更可靠，但也更"小家子气"。

### 4. 跨界秩序（凡间 / 灵界 / 仙界）

- 三界灵气浓度递增（稀薄 → 浓郁 → 液态）；
- 主流修行体系各异（道/佛/儒/魔 → 剑修/丹道/阵道 → 法则/神通）；
- **飞升必须经灵界中转**，不允许凡间直接跨到仙界；
- 仙界不是终点，是**更卷的下一层**。NPC 仙人背后有"仙界世家""仙门道统"，照样有派系倾轧。
- 详细三界规则已写在 `three-realms-detail.md`，本文件不重复。

## 四、人情世故与价值尺度（**AI 最容易写假的部分**）

这一节是给 AI 的"反现代人味儿"约束。AI 写修仙者最容易出的问题，是把角色写成"现代人套修仙皮"。要避开这些：

### 1. 弱肉强食是常态，不是禁忌

- 杀人不眨眼是常态，灭门屠城不止一次。
- 元婴期老魔动辄灭一国，不是夸张。
- 反例写法：一笑泯恩仇、师徒如父子、闺蜜情深 —— 大多数情况下都不真实。

### 2. 双修与道侣

- 双修是"道侣之间互相促进修为"，**不是现代意义的恋爱**。
- 元婴期的双修对象通常也是元婴；筑基期和元婴期双修等于"被采补"。
- 例：《凡人修仙传》中韩立与南宫婉的关系是"道侣"而非"恋人"，双方修为互相支持，聚少离多。

### 3. 师徒关系

- 师父收徒多半看资质，徒弟叛师时有发生。
- "一日为师终身为父"在大多数修仙世界里**不是通行规则**。

### 4. 仇杀与化干戈

- 化干戈为玉帛极少发生，需要双方都是利益交换。
- 例：《凡人修仙传》中韩立的仇人通常被杀干净或打到不敢现身。

### 5. 灵兽灵虫的"通灵"关系

- 灵兽不是工具，是生命。
- 灵兽养久了会通灵，能听懂简单的指令，但不会说话。
- 高级灵兽可化形为人，越阶强行收服顶级仙禽**几乎必然失败**。
- 例：《凡人修仙传》中韩立与噬金虫、啼魂兽的关系是"伙伴"而非"宠物"。

## 五、时间跨度与"宗门世"叙事节奏

这是修仙叙事的**时间感**，AI 写"修仙"如果一年半载就"飞升"，整个味道就崩了。

### 1. 凡人 vs 修士的时间观

- 凡人用"年"算日子，修仙者用"甲子""百年"算。
- "百日筑基"是民间讹传。原作中筑基是无数散修一辈子跨不过去的门槛。

### 2. 跨大境界的时间尺度（《凡人修仙传》原作可考）

| 阶段 | 时长 |
|------|------|
| 炼气 → 筑基 | 数十年 |
| 筑基 → 结丹 | 数百年 |
| 结丹 → 元婴 | 数百年至千年 |
| 元婴 → 化神 | 上千年 |
| 化神 → 飞升 | 上千年 |
| 飞升灵界时已活了多久 | 上千年以上 |

### 3. 宗门换代 / 世家代际

- 大型宗门每隔数百年会"换代" —— 老一代坐化或飞升，新一代崛起。
- 凡人世家百年换代，修仙家族数百年换代。
- 修仙者看着一代又一代同辈坐化 —— 这是修仙小说的核心孤独感，AI 不要回避。

## 六、战斗 / 副职业的真实份量

修仙小说的"实力"不是单一的"修为数字"，是**多维度组合**：

### 1. 五大副职业的真实份量

- **炼丹**：决定修士"上限" —— 结丹、化神、渡劫都需对应丹药辅助。高阶丹方本身就是稀缺资源。
- **炼器 / 本命法宝**：本命法宝是修士的"半条命"，从祭炼、温养、渡劫到器灵觉醒是一条长线。
- **阵法**：既是宗门护山大阵、也是战斗杀阵、更是困敌/困己的核心；高阶阵法可困一域。
- **符箓**：是低阶修士最实用的"越阶工具"，也是符修一道的根基；高级符箓可伤化神、封印元婴。
- **傀儡**：可做替身、分身、探子；顶级傀儡甚至能承载部分道则。

详细规则已在 `formations-restrictions.md` / `alchemy-handfeel.md` 等专题，本文件不重复。

### 2. 本命法宝的铁律

- 用自身精血长期温养，与主人心神相连。
- 威力随主人修为增长而增长。
- 一旦主人陨落，本命法宝往往也损毁（除非提前解除）。
- 本命法宝被人毁，等于断道。

## 七、AI 写作时的几条铁律（横切约束）

这一节是把上面的常识收成可执行的"AI 写法禁令"，喂 prompt 时直接用。

### 1. 不要写成现代人套修仙皮

- 修仙者不说"加油""努力""把握机会"这种现代口癖。
- 修仙者不说"我真的很喜欢这份工作""和你在一起很开心"。
- 修仙者不动不动"感动""感恩""致敬"。
- 例外：低龄（0-6）幼童可以懵懂，但大人不该有这种现代情绪。

### 2. 力量刻度不能乱

- 炼气期不飞、不瞬移、不撕裂虚空。
- 元婴期才是大部分高阶战斗的起点。
- 化神期才是人界天花板。
- 越阶战斗必须有"底牌 + 环境 + 代价"三件套。

### 3. 不要在低阶主角身上堆神级设定

- 凡人家庭出身的幼童不该"觉醒了前世记忆"（除非是上一世重生的明确剧情锚）。
- 散修出身不该一上来就"识海里住着老爷爷"。
- 例外：若剧情已埋好的传承/夺舍/奇遇，按剧情走；但 AI 不能擅自添加。

### 4. 不要把修仙世界写成公平社会

- 血脉世家、宗门、师承是常态；散修是被压迫的弱势方。
- 资源分配不公、机会不平等，是修仙小说反复渲染的"真实感"，不要回避。
- 宗门任务、派系倾轧、长老会议、嫡庶之争是日常。

### 5. 时间跨度不打折

- 不写"一年筑基""十年飞升"。
- 不写凡人视角的修仙者用"日""月"为单位计算重大事件。
- 修仙者用"甲子""百年"。

### 6. 死亡不是终点的写法要克制

- 修士死后仍有真灵、魂魄、转世、夺舍多条路可走；
- 但"死了就真死"反而是少数派 —— 大多数核心角色死不了，重要配角才会真正消失。
- 写主角不能动不动"献祭""魂飞魄散"，否则后续剧情接不上。

### 7. 不要把世界观讲成"哲学启示"

- AI 写修仙剧情时，不要在叙事中插入"启示录""寓言""道德教训"。
- 修仙世界的"道理"是通过事件和人物行动呈现，不通过旁白和说教。

## 八、本文件的使用方式（提示词注入点）

- **system-identity 区（恒定身份）**：本文件不直接注入。这里的"硬规则"是游戏逻辑约束（如"不写玩家光环""不外暴露机制词"）。
- **scene-behavior 区（按场景加载）**：本文件按"灵根／境界／宗门／秘境"等槽位在 prompt-zones 中钩子调用，最大段不超过 1500 字。
- **retrieved-memories 区（长期记忆）**：本文件按"近期剧情涉及的世界观关键词（天道／劫数／灵脉／世家／灵兽 等）"做 RAG 检索，召回片段后再按 token 预算截取。

实际注入由 `src/lib/xianxia/prompt-zones.ts` + `rag/prompt-augmentor.ts` 控制；本文只负责内容沉淀，**不引入新的硬编码 token 流**。
