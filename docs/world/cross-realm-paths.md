# 跨域通道与星空 (L1-7)

> 修仙世界的空间结构。域 = 区域；跨域 = 跨区域移动；星空 = 三界之外的广袤空间。
>
> **AI 接管**：跨域通道由 AI 据角色境界 + 因果决定可走哪条；星空事件由 AI 自由创作。

---

## 1. 域的划分

### 1.1 凡界中的域

| 域 | 内部 key | 范围 | 主要势力 |
|------|------|------|------|
| 东域 | `east` | 东部大陆 | 青云门 / 万剑宗 |
| 南域 | `south` | 南部海岛 | 碧波宫 / 散修联盟 |
| 西域 | `west` | 西部荒漠 | 沙陀寺 / 游牧散修 |
| 北域 | `north` | 北部冰原 | 玄冰宫 / 雪域散修 |
| 中域 | `central` | 中央神山 | 三大圣地 |
| 海外 | `overseas` | 远海仙山 | 蓬莱 / 方丈 / 瀛洲 |
| 幽冥 | `netherworld` | 地下 / 死后世界 | 轮回司 |

### 1.2 域的特色

- **东域**：剑修为主，灵气浓郁
- **南域**：商贸发达，岛屿散落
- **西域**：佛教 / 魔修并存
- **北域**：妖族聚居，冰修为主
- **中域**：仙缘最重，传说纷起
- **海外**：化外之地，远古遗留
- **幽冥**：死后 / 阴修之所

## 2. 跨域通道

### 2.1 通道类型

| 通道 | 内部 key | 速度 | 限制 |
|------|------|------|------|
| 陆路 | `land` | 慢 | 必经之城 |
| 海路 | `sea` | 慢 | 需海图 / 抗晕船 |
| 飞剑 | `sword` | 中 | 需元婴+ |
| 传送阵 | `teleport` | 极快 | 需高品级阵 + 巨资 |
| 虚空裂缝 | `voidRift` | 不可控 | 危险 |
| 星空古路 | `starPath` | 跨三界 | 需大乘+ |

### 2.2 风险等级

| 通道 | 风险 | AI 事件 |
|------|------|------|
| 陆路 | 低 | 路遇散修 / 野兽 |
| 海路 | 中 | 海兽 / 风暴 |
| 飞剑 | 中 | 罡风 / 迷路 |
| 传送阵 | 低 | 偶有错位 |
| 虚空裂缝 | 极高 | 不可预知 |
| 星空古路 | 高 | 星辰兽 / 迷路 |

## 3. 星空

### 3.1 星空结构

| 区域 | 内部 key | 特征 |
|------|------|------|
| 内星空 | `innerSky` | 三界之间，相对安全 |
| 外星空 | `outerSky` | 广袤无垠，未知 |
| 星辰 | `stars` | 移动的"岛"，可登陆 |
| 星海 | `starSea` | 灵气潮汐
| 虚空 | `void` | 一切法则失效 |

### 3.2 星空生物

- 星辰兽：生于星海的灵兽
- 虚空游者：跨域的流浪者
- 星灵：星空的本源意识
- 失仙：被贬入星空的仙人

### 3.3 星空资源

- 星辰铁：炼器极品
- 星辉：阵法极品
- 虚空石：跨域通道材料
- 时光沙：禁制核心材料
