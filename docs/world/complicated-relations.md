# 亦敌亦友与忘年交 (L1-8)

> 修仙世界的关系不是非黑即白。亦敌亦友、忘年交是典型的复杂关系。
>
> **AI 接管**：复杂关系由 AI 据角色过往 + 当前事件动态评估，引擎只负责持久化。

---

## 1. 关系维度

### 1.1 多元关系

单一关系不足以描述修仙世界。关系至少有 5 个维度：

| 维度 | 内部 key | 范围 | 说明 |
|------|------|------|------|
| 亲疏 | `intimacy` | -100~100 | 心理距离 |
| 信任 | `trust` | 0~100 | 是否能托付 |
| 利益 | `interest` | -100~100 | 利益冲突 |
| 恩义 | `favor` | 任意值 | 受恩 / 结仇 |
| 立场 | `stance` | enum | 同盟 / 中立 / 对立 |

### 1.2 关系组合

| 组合 | 玩家可见 label | 典型情况 |
|------|------|------|
| 亲 + 信 + 同盟 | 至交 | 多年好友 |
| 亲 + 利 | 挚友有别 | 利益纠葛的挚友 |
| 亲 + 立 | 知己对立 | 立场不同但互相欣赏 |
| 疏 + 信 + 同盟 | 战友 | 因事结盟 |
| 疏 + 仇 | 仇敌 | 不可调和 |
| 0 + 0 + 同盟 | 同道 | 仅为目标合作 |

---

## 2. 亦敌亦友

### 2.1 定义

双方在某些维度对立（立场、利益），在另一些维度亲近（亲密度、信任）。

### 2.2 典型场景

- **棋逢对手**：双方争夺某物，但互为对手也是知己
- **道不同**：立场对立但彼此敬重
- **宿敌成友**：曾为敌，因某事理解彼此
- **同门操戈**：同一宗门内部派系斗争
- **相爱相杀**：情 + 仇的复杂关系

### 2.3 AI 接管

- **关系演化**：AI 据事件叙事让关系多维度变化
- **叙事张力**：AI 写"仇人相见"或"挚友反目"的张力
- **关系展示**：UI 用多维度色块呈现

---

## 3. 忘年交

### 3.1 定义

年龄 / 辈分差距巨大但精神契合的友谊。

### 3.2 典型场景

- **师徒之外**：不正式拜师，但传功解惑
- **忘年知己**：年长者视年幼者为友
- **童颜老妖**：妖族外表年幼但实为千年老妖
- **少年宗师**：天赋异禀的少年与老修士
- **前世今生**：前世是师徒，今生为同辈

### 3.3 AI 接管

- **关系判定**：AI 据年龄差 + 心性契合度判定是否"忘年"
- **关系叙事**：AI 写"年长者不居高临下"或"年少者不幼稚"的具体场景
- **冲突处理**：AI 据"忘年"特性决定冲突如何收场

## 4. 其他复杂关系

### 4.1 师徒之外的复杂关系

| 关系 | 描述 |
|------|------|
| 半师半友 | 师承有别但形同友人 |
| 护道者 | 保护但不传授 |
| 故人转世 | 前世是友人，今生重逢 |
| 神交 | 未见面但精神共鸣 |

### 4.2 跨辈分 / 跨种族 / 跨界关系

- **跨辈分**：师祖与徒孙为友
- **跨种族**：人与妖的忘年交
- **跨界**：凡界与灵界飞升前后为友

## 5. AI 接管

- **关系复杂度评估**：AI 据事件复杂度判定关系标签
- **关系叙事**：AI 写"何以至此"的复杂情感
- **关系演化**：AI 据"长期互动"动态调整多维度数值
