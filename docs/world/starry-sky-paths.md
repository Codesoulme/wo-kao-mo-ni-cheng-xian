# 星空通道

> L2 数据契约文档（2026-06-27）。描述跨界通道的具体形式与规则。

## 通道类型

| 类型 | 描述 | 难度 | 成本 |
|------|------|------|------|
| 飞升 | 通过天劫从低界升至高界 | 80-95 | 0 |
| 星空 | 灵界→凡间的高消耗通道 | 60 | 100000 灵石 |
| 仙令 | 仙界→任意界（需持有令牌） | 40-70 | 0 |
| 禁行 | 剧情锁定的单向通道 | - | - |

## 通道列表

### 升界通道
- 凡间 → 灵界：飞升（渡天劫）
- 灵界 → 仙界：飞升（渡更强天劫）

### 降界/跨界通道
- 灵界 → 凡间：星空通道（高消耗）
- 仙界 → 灵界：仙令（需持有）
- 仙界 → 凡间：仙令（需持有 + 修为压制）

### 禁行通道
- 凡间 → 仙界：直接跨界不允许

## 通道代价

| 通道类型 | 灵石消耗 | 修为折损 | 寿命折损 |
|---------|---------|---------|---------|
| 飞升 | 0 | 0 | 0 |
| 星空 | 100000 | 30% | 50 年 |
| 仙令 | 0 | 50% | 100 年 |

## 通过条件

- **飞升**：境界达标 + 天劫已渡 + 飞升资格齐全
- **星空**：修为 ≥ 元婴期 + 灵石 ≥ 100000
- **仙令**：持有"仙令"道具（极稀有）
