# 阵法与禁制 (L1-6)

> 阵法是利用灵气/符文在空间中构造的力量；禁制是对人/物/地的限制机制。
>
> **AI 接管**：阵法由 AI 据角色技能 + 资源设计；禁制由 AI 据剧情需要创建。

---

## 1. 阵法基础

### 1.1 阵法要素

| 要素 | 内部 key | 作用 |
|------|------|------|
| 阵旗 | `flags` | 固定阵法位置 |
| 阵图 | `formation` | 阵法蓝图 |
| 阵眼 | `eye` | 阵法核心 |
| 灵力 | `spirit` | 启动能源 |
| 主持者 | `controller` | 维持阵法 |

### 1.2 阵法分类

| 类型 | 内部 key | 作用 |
|------|------|------|
| 防御阵 | `defense` | 抵御外敌 |
| 困阵 | `trapping` | 困住敌人 |
| 杀阵 | `killing` | 杀伤敌人 |
| 幻阵 | `illusion` | 制造幻象 |
| 聚灵阵 | `gathering` | 加速修炼 |
| 传送阵 | `teleport` | 跨域传送 |
| 结界阵 | `barrier` | 隔绝天地 |
| 反阵 | `counter` | 克制其他阵 |

## 3. 禁制体系

### 3.1 禁制类型

| 类型 | 内部 key | 作用 | 解除方式 |
|------|------|------|------|
| 封印禁 | `seal` | 封印力量 / 记忆 / 修为 | 特定手法 / 钥匙 |
| 锁灵禁 | `lockSpirit` | 封锁灵气流动 | 破坏阵眼 |
| 心神禁 | `mindLock` | 控制心智 | 渡心魔 |
| 诅咒禁 | `curse` | 持续伤害 | 化解咒源 |
| 时空禁 | `timeSpace` | 禁锢时空 | 极难 |
| 血脉禁 | `bloodline` | 绑血脉 | 血脉断绝 |

### 3.2 禁制来源

- 强者遗留
- 宗门守护
- 个人诅咒
- 天道反噬
- 异宝自带

---

## 4. AI 接管

- **阵法设计**：AI 据角色 + 场景设计专属阵法
- **禁制创作**：AI 据剧情需要创建"为什么有这个禁制"
- **解谜逻辑**：AI 写"如何破阵/解禁"
- **副作用**：AI 据"强行破解"评估后果

---

## 5. 破阵机制

### 5.1 破阵方式

| 方式 | 难度 | 风险 |
|------|------|------|
| 找到阵眼 | 中 | 触发陷阱 |
| 暴力破阵 | 高 | 阵法反噬 |
| 巧解 | 中 | 需阵法知识 |
| 寻人解阵 | 低 | 需人情 |
| 等待失效 | 0 | 时间长 |

### 5.2 反破阵

- 阵法自动防御
- 阵眼隐匿
- 阵中套阵
- 触发禁制
