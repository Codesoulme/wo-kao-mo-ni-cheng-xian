# 跨域 NPC

> L2 数据契约文档（2026-06-27）。描述三界 NPC 的分层与跨域规则。

## 三界 NPC 分层

| 层级 | NPC 类型 | 数量 | 修为范围 |
|------|---------|------|---------|
| 凡间 | 凡人/散修/低阶弟子 | 100+ | 炼气 → 化神 |
| 灵界 | 高阶散修/化形灵兽/跨界商人 | 60+ | 化神 → 渡劫 |
| 仙界 | 仙人/仙兽/神祇后裔 | 20+ | 渡劫+ |

## 跨域通行权

- **持有者**：跨界商人、飞升失败滞留者、特定剧情 NPC
- **获取方式**：
  - 飞升失败（滞留灵界可返回凡间）
  - 完成特殊任务（仙令、跨界令牌）
  - 特殊身份（神祇后裔天生持有）
- **限制**：
  - 下界修为压制 50%
  - 跨界消耗大量灵石（10w+）
  - 部分通道限时/限次

## NPC 与角色互动

- 三界 NPC 默认不互通，需要通过通道或剧情触发
- 通道由 `worldTier` + `crossRealmAccess` 共同决定可访问性
- 角色所在层级决定可遭遇哪些 NPC
