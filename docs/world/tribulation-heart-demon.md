# 天劫与心魔 (L1-3)

> 修仙世界两大关键障碍。天劫是外部天道的考验；心魔是内部的执念与欲望。
>
> **AI 接管**：心魔叙事由 AI 据角色过往 + 执念自由创作；天劫由 AI 据境界自由设计。

---

## 1. 天劫体系

### 1.1 劫难分级

| 劫级 | 内部 key | 触发境界 | 强度 |
|------|------|------|------|
| 雷劫 | `thunder` | 筑基→金丹 | ×1 |
| 风火劫 | `windFire` | 金丹→元婴 | ×3 |
| 三灾 | `threeCalamities` | 元婴→化神 | ×5 |
| 心魔劫 | `heartDemon` | 化神飞升 | ×8 |
| 九九天劫 | `nineNine` | 大乘破界 | ×15 |
| 仙劫 | `immortal` | 灵界→仙界 | ×30 |

### 1.2 渡劫机制

- 渡劫期间禁止外援（天道考验，外部干预视为作弊）
- 玩家可选：硬抗 / 法宝 / 阵法 / 避劫（后果各异）
- 失败 = 退境 + 严重伤害（少数情况灰飞烟灭）

### 1.3 AI 接管

- **劫难定制**：AI 据角色灵根 + 心魔 + 执念设计专属劫难
- **渡劫叙事**：AI 写"天雷轰下"的具体画面
- **渡劫奖励**：AI 据"渡劫时心境"决定额外奖励

## 2. 心魔体系

### 2.1 心魔类型

| 类型 | 内部 key | 来源 | 难度 |
|------|------|------|------|
| 贪心魔 | `greed` | 求财 / 求宝 | 低 |
| 嗔心魔 | `anger` | 仇怨 / 愤怒 | 中 |
| 痴心魔 | `delusion` | 执念 / 痴迷 | 高 |
| 慢心魔 | `pride` | 自负 / 傲慢 | 中 |
| 疑心魔 | `doubt` | 不信 / 动摇 | 高 |
| 悲心魔 | `sorrow` | 失去 / 悔恨 | 中 |
| 恐心魔 | `fear` | 畏惧 / 怯懦 | 低 |
| 爱别离 | `separation` | 失爱 / 离恨 | 高 |

### 2.2 心魔累积

- 每次负面情绪事件 +1-3 心魔
- 杀生 +5
- 背叛信任 +10
- 心魔 ≥ 50 → 触发心魔试炼
- 心魔 ≥ 80 → 走火入魔风险
- 心魔 = 100 → 永久入魔

### 2.3 心魔化解

| 方式 | 效果 | 副作用 |
|------|------|------|
| 闭关冥想 | -10 | 失去 5 年 |
| 与人谈心 | -15 | 暴露心事 |
| 心魔试炼 | -30 | 失败 +20 |
| 舍身救人 | -50 | 可能身死 |
| 顿悟 | -80 | 境界突破 |

### 2.4 AI 接管

- **心魔叙事**：AI 据角色过往事件写"心魔化形"的试炼关卡
- **试炼关卡**：AI 自由设计（如：旧仇人显化 / 父母重逢 / 爱人诀别）
- **化解路径**：AI 据角色性格判定"哪种化解最合理"

---

## 3. 心魔与天劫的联动

- 心魔高 → 天劫难度 +50%
- 心魔试炼失败 → 渡劫时心魔劫自动失败
- 心魔化解成功 → 渡劫时获得额外保护
