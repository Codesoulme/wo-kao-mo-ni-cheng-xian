# 三界详情

> L2 数据契约文档（2026-06-27）。详细描述凡间/灵界/仙界三个层级的设定。

## 凡间（humanWorld）

- **灵气浓度**：稀薄
- **主流修行体系**：道、佛、儒、魔
- **特色**：宗门林立，凡人与修士共存；资源以矿脉/灵草为主
- **NPC 类型**：凡人、散修、低阶宗门弟子
- **秘境**：洞天福地（5 个）+ 古修士遗迹（10 个）

## 灵界（spiritWorld）

- **灵气浓度**：浓郁
- **主流修行体系**：剑修、丹道、阵道
- **特色**：高阶修士聚集；飞升失败者滞留此界；与凡间有星空通道
- **NPC 类型**：高阶散修、灵兽化形、跨界商人
- **秘境**：星辰废墟（12 个）+ 跨界通道（3 条）

## 仙界（immortalWorld）

- **灵气浓度**：液态
- **主流修行体系**：法则、神通
- **特色**：仙人居所；时间流速与凡间不同；持有"仙令"可下界
- **NPC 类型**：仙人（极少）、仙兽、神祇后裔
- **秘境**：仙人遗迹（5 个）+ 法则试炼场（3 个）

## 跨界规则

- **凡间 → 灵界**：渡天劫飞升
- **灵界 → 仙界**：再渡一次更强天劫飞升
- **灵界 → 凡间**：星空通道（消耗大量灵石 + 折损修为）
- **仙界 → 凡间**：需"仙令"，且每次下界修为压制 50%
- **凡间 → 仙界**：不允许直接跨越，必须经灵界中转
