# 炼丹手感与火候 (L1-5)

> 炼丹不只是"投入材料"，是手感、时机、心境的综合。
>
> **AI 接管**：炼丹火候由 AI 据"此刻意境"评估，决定丹药品质与副作用。

---

## 1. 炼丹要素

| 要素 | 内部 key | 影响 |
|------|------|------|
| 火候 | `heatLevel` | 决定药性激活度 |
| 时机 | `timing` | 决定药力融合度 |
| 心境 | `mindset` | 决定额外属性 |
| 手法 | `technique` | 决定失败率 |
| 灵草配比 | `herbRatio` | 决定丹方契合度 |

### 1.1 火候等级

| 火候 | 内部 key | 中文 label | 适用丹药 |
|------|------|------|------|
| 微火 | `low` | 微火 | 养气丹 / 筑基丹 |
| 中火 | `medium` | 中火 | 金丹辅助丹 |
| 武火 | `high` | 武火 | 元婴突破丹 |
| 文武火 | `wenhigh` | 文武火 | 化神辅助丹 |
| 三味真火 | `samadhi` | 三味真火 | 仙品丹 |

### 1.2 时机

- 入药时机（每株灵草何时投入）
- 转火时机（火候转换的精确时间点）
- 收丹时机（最关键的几息内）

## 2. 炼丹手感

### 2.1 什么是"手感"

- **手感好**：炼丹师对火候/时机/配比的直觉
- **手感差**：机械操作，凭丹方不知变通
- **手感具象化**：AI 写"炉中药香在此时变化""火苗偏了半寸"等具体感受

### 2.2 手感来源

| 来源 | 效果 |
|------|------|
| 反复练习 | 稳定 +5% 成功率 |
| 师承 | 起始 +15% 成功率 + 独家丹方 |
| 顿悟 | +30% 成功率 + 解锁隐藏丹方 |
| 心魔 | -20% 成功率 + 走火风险 |
| 心境平静 | +10% 品质 |

### 2.3 AI 接管

- **手感叙事**：AI 写"此刻你感觉炉中灵气流向"
- **火候判定**：AI 据角色状态 + 丹方决定火候曲线
- **时机判定**：AI 写"何时投入最后一味药"
- **失败/副作用**：AI 据"手感偏移"决定走火后果

## 3. 丹药品质与火候关系

| 火候精准 | 丹品 | 副作用概率 |
|------|------|------|
| 95%+ | 绝品 | 0% |
| 85-95% | 极品 | 2% |
| 70-85% | 上品 | 5% |
| 50-70% | 良品 | 10% |
| 30-50% | 凡品 | 20% |
| <30% | 失败 | 100% |

### 3.1 副作用

- 丹毒（凡品常见）：服药后 +5 心魔
- 药性冲突（混合炼时）：走火风险
- 隐性副作用：需特殊手法探查

- 失败：「药性反冲，炉鼎炸裂」
