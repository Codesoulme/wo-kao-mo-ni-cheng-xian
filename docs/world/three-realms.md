# 三界飞升设定 (L1-2)

> 修仙世界的空间结构与飞升机制。下界 → 灵界 → 仙界的三层结构。
>
> **AI 接管**：飞升路径由角色决策 + 因果 + 灵根决定，AI 据此写飞升叙事。

---

## 1. 三界结构

| 界 | 中文 label | 特征 | 灵气浓度 | 寿元上限 | 飞升条件 |
|------|------|------|------|------|------|
| 凡界 | 凡尘 | 凡人 + 低阶修士 | ×1 | 500 岁 | 化神期 + 渡劫 |
| 灵界 | 灵域 | 中高阶修士 | ×10 | 5000 岁 | 大乘期 + 破界 |
| 仙界 | 仙台 | 真仙 | ×100 | 永生 | 渡仙劫（剧情触发） |

### 1.1 飞升层级图

```
凡界 (炼气 → 筑基 → 金丹 → 元婴 → 化神)
   │   化神期渡天劫 → 飞升
   ↓
灵界 (化神 → 炼虚 → 合体 → 大乘)
   │   大乘期破界壁 → 飞升
   ↓
仙界 (真仙 → 金仙 → 大罗金仙 → 道祖)
```

## 2. 飞升机制

### 2.1 化神渡劫

| 劫 | 内部 key | 难度 | 失败后果 |
|------|------|------|------|
| 心魔劫 | `heartDemonTribulation` | 中 | 心魔 +20 |
| 天雷劫 | `thunderTribulation` | 高 | 气血 -30% |
| 三灾劫 | `threeCalamities` | 极高 | 失败 = 飞升失败 + 退境 |
| 九天雷劫 | `nineHeavens` | 极难 | 失败 = 灰飞烟灭 |
| 情劫 | `loveTribulation` | 中 | 取决于角色 |

### 2.2 灵界破界

- 需大乘期 + 5 缕先天灵气
- 失败 = 退到化神巅峰 + 50 年冷却

### 2.3 仙界渡仙劫

- 需破界后 + 100 年灵界修整
- 失败 = 退回灵界

## 3. AI 接管

- **劫难设计**：AI 据角色状态设计专属劫难
- **飞升叙事**：AI 写"渡过此劫，踏入新界"（100-200 字）
- **新界设定**：AI 自由设定灵界/仙界具体地理、势力、规则
- **飞升失败叙事**：AI 写"渡劫未成"的痛苦与恢复
