# 飞升流程

> L2 数据契约文档（2026-06-27）。描述《我靠模拟成仙》中三界飞升的完整流程。

## 三界层级

| 层级 | 名称 | 起点境界 | 飞升目标 |
|------|------|---------|----------|
| 凡间 | humanWorld | 炼气 → 大乘 | 灵界 |
| 灵界 | spiritWorld | 大乘 → 渡劫 | 仙界 |
| 仙界 | immortalWorld | 渡劫 → ? | 不再飞升 |

## 飞升要求

### 凡间 → 灵界
- 最低境界：大乘期（mahayana）
- 天劫：必须已渡
- 寿命：≥ 500 岁
- 声望：≥ 5000
- 修为：≥ 100000
- 道心强度：≥ 80

### 灵界 → 仙界
- 最低境界：渡劫期（ascension）
- 天劫：必须已渡
- 寿命：≥ 2000 岁
- 声望：≥ 50000
- 修为：≥ 1000000
- 道心强度：≥ 95

## 飞升触发条件

- 大乘期角色年满 500 岁自动触发飞升入口
- 渡劫期角色年满 2000 岁自动触发飞升入口
- 玩家可通过 UI 主动尝试（需先满足资格）

## 飞升判定

- 基础阈值 0.5 + 道心加成（daoHeart / 200）
- 天劫未渡直接失败
- 飞升失败跌回原境（不致死，可重试）

## 飞升后果

- **成功**：进入新界，境界提升 + 视野解锁新 NPC/秘境/资源
- **失败**：跌回原境 + 道心小幅受损
- **放弃**：保留原境但失去本次飞升机会（冷却 100 年）
