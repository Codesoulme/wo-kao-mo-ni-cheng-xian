# 《我靠模拟成仙》事件蓝图（DESIGN-EVENT-BLUEPRINTS）

> 文档版本：v0.1
> 状态：phase-g 设计细化稿（第一版）
> 适用：`src/lib/xianxia/types.ts` 的 `BlueprintCategory`（cultivation / encounter / social / combat / trade / exploration / heritage / trial / emotion / inner_demon / thread_resolve / daily）
> 关联：本表按**人生阶段 / 事件类型**组织（出生 / 童年 / 入门 / 突破 / 战斗 / 拍卖 / 秘境 / 因缘了结 / 重伤 / 渡劫 / 飞升），每类给出触发条件、状态影响、可能战利品、推荐行动、典型叙事钩子、≥3 个示例
> 与 `docs/blueprints/event-blueprint.md` 的关系：本文档**补充人生阶段维度**，不替换事件类型路由表
> 与 `docs/blueprints/value-blueprint.md` 的关系：本表遵循价值取向，所有"获得/失去"必须在价值取向范围内

## 1. 出生（birth）

### 1.1 触发条件

- 玩家新建角色 / 重置世界 / 继承上一代结算
- `status.id == 'init'` 且 `age == 0`
- AI 必须基于随机灵根 + 五行倾向 + 出身背景生成出生叙事

### 1.2 状态影响

- 灵根定型（不可变更）
- 出生家族身份（如：修仙世家 / 散修孤儿 / 凡人农户）
- 起始修炼速度倍率定型（按灵根决定）
- 起始因缘 1~2 条（家族血脉 / 天降异象）

### 1.3 可能战利品 / 修炼加成 / 减益

- 战利品：家族信物（如有）1 件；无附加修炼加成
- 加成：仙缘（极小概率）：开局 +5% 修炼速度
- 减益：异灵根 → 修炼速度 -10%；杂灵根 → 0%

### 1.4 推荐行动（AI 参考）

- 写入"出生即非凡"的伏笔（如有）；不要立刻给大规模事件
- 写一段 100~150 字的出生叙事 + 1~2 个后续可承接的钩子
- 与家族 / 母亲 / 族人建立初始关系值（30~60）

### 1.5 典型叙事钩子

- 母亲产前夜梦见仙人托梦
- 接生婆见婴儿掌心有异纹
- 同村井水忽然泛起灵光
- 远山雷云骤然散去

### 1.6 示例（≥3）

**示例 1（出生即带仙缘）**：母亲难产，夜中忽有金光入室，产下天灵根之子。父惊讶，亲族以为是吉兆，命名"玄明"。
**示例 2（散修孤儿）**：生母为游方女修，产子后力竭而亡，父不明，婴孩被路过的散修拾得抚养。
**示例 3（异象引祸）**：婴儿出生时雷云压顶三日不散，村人惧为妖孽，私下劝父弃之，母不忍。

## 2. 童年（childhood）

### 2.1 触发条件

- `age ∈ [0, 12]`（参见 `EVENT_BLUEPRINTS` 中 `minAge`/`maxAge`）
- 玩家连续推进 + AI 调度日场事件
- 与 `BlueprintCategory` 的 `daily` / `encounter` / `social` / `inner_demon` 重叠

### 2.2 状态影响

- 多为日场（`daily`），不主动施加 buff/debuff
- 可获得童年执念 / 心结（影响成年心境 → `inner_demon` 来源之一）
- 与家人、玩伴建立初始关系值（30~70）

### 2.3 可能战利品 / 修炼加成 / 减益

- 战利品：玩具 / 草木 / 小兽
- 加成：童蒙启智（悟性 +1%，极罕见）
- 减益：童年执念（未来心魔来源之一）

### 2.4 推荐行动（AI 参考）

- 严格按"幼龄规则"：0~6 岁不能独立赴约、整理行装、执行约定、追查因缘、主动探索
- 童年叙事应以"在大人看护下玩耍 / 听闻传说 / 被家人带去 / 懵懂盼头"为主
- 长度 80~120 字；不要套用成人线索承接模板

### 2.5 典型叙事钩子

- 邻居爷爷讲古"三百年前那只白狐"
- 与玩伴偷偷跑去后山（被大人抓回）
- 母亲病重，婴孩守在床边
- 山洪冲毁半村，亲人失踪

### 2.6 示例（≥3）

**示例 1**：夏日傍晚与邻家小儿在溪边捉蟹，归家时母亲已备好饭团，父亲笑说"再不回来就要化妖了"。
**示例 2**：七岁那年冬夜，村中老猎户讲起山上白额虎的故事，讲到一半忽压低声音："那头虎已成精了"。
**示例 3**：十岁时母亲病重，半夜婴孩守在床边，听见母亲梦中呢喃"洞府……小镜……"。

## 3. 入门（initiation）

### 3.1 触发条件

- 拜师 / 入宗 / 偶得传承
- `master != null || faction != null || pendingThread.category == 'inheritance'`
- 触发后 `BlueprintCategory = 'cultivation' || 'heritage'`

### 3.2 状态影响

- 获得身份（identity.sect_inner_disciple / registered_disciple / 等）
- 获得起始功法 1~2 本
- 修炼速度基数定型
- 因缘"师徒之约"或"宗门荣辱"建立

### 3.3 可能战利品 / 修炼加成 / 减益

- 战利品：信物（内门令牌 / 师尊遗物 / 入门功法）
- 加成：内门身份 +5%；师尊传功（+15% 短期）
- 减益：身份束缚（不可叛宗出逃 / 任务强制）

### 3.4 推荐行动（AI 参考）

- 入门场景必须有完整的拜师礼 / 入门礼 / 试炼场景
- 写入门那一刻的情绪（激动 / 惶恐 / 平静）
- 与师尊或宗门长老建立首条 `relationships` 关系值（50~70）

### 3.5 典型叙事钩子

- 师尊的叩问："你为何入我门下？"
- 外门师兄的冷眼与刁难
- 入门试炼中救下未来道侣
- 在祠堂中看见上一代陨落弟子的牌位

### 3.6 示例（≥3）

**示例 1（拜师）**：十二岁那年被青岚宗外门长老看中收入门下，拜师大典上师尊只问一句"为何修仙"，孩童答"为母亲延寿"，师尊点头不语。
**示例 2（入宗）**：十三岁通过青岚宗外门试炼，在演武场上击败三头木傀儡，外门师兄递来一枚铜牌："外门弟子，今日起算。"
**示例 3（散修偶得）**：十四岁独自进山采药，误入一处荒废洞府，得玉简一枚，载残缺功法《吐纳引》。

## 4. 突破（breakthrough）

### 4.1 触发条件

- `cultivationExp >= expPerLevel` 且 `level < realm.maxLevel`
- 或 `realm == realm.lower && pendingThread.category == 'breakthrough'`
- 由引擎校验 + AI 写突破叙事（参见 DESIGN-VALUES.md §3）

### 4.2 状态影响

- 境界变更（realm / level）
- 可能获得新境界特性（`realmTraits`）
- 突破成功 → 寿元上限刷新（+境界对应值）
- 突破失败 → 扣气血 / 寿元 / 加心魔

### 4.3 可能战利品 / 修炼加成 / 减益

- 战利品：突破丹 / 灵液消耗（视玩家行为）
- 加成：境界小层级加 +0.5% 修炼速度
- 减益：失败时按境界触发（炼气失败扣气血 / 金丹失败扣寿元）

### 4.4 推荐行动（AI 参考）

- 突破前必须有"准备 → 冲关 → 临门一脚 / 失败 / 成功"四段叙事，不能一蹴而就
- "破境 / 突破"标签只在**最终成功**叙事上显示，准备/失败/临门不能提前显示
- 突破失败时必须给出后续处置（疗伤 / 调息 / 重整旗鼓）
- 雷劫 / 渡劫单独走 §10

### 4.5 典型叙事钩子

- 师尊闭关前最后一次点化
- 突遇外敌袭击，借压破势强冲
- 失败后遇到昔日故人
- 突破瞬间天地异象（如有）

### 4.6 示例（≥3）

**示例 1（炼气→筑基）**：十八岁那年闭关三月，丹药齐备、心境圆融，冲关至第九日晨，灵台轰然一震，筑基成。师尊在门外点了点头。
**示例 2（筑基失败）**：二十二岁筑基失败，灵台受损，气血跌至三成，被师尊搀出闭关室，嘱"再休三年"。
**示例 3（金丹→元婴）**：一百二十岁，金丹九转圆满，于某夜独坐洞府，忽然紫气东来，元婴破丹而出，识海一片清明。

## 5. 战斗（combat）

### 5.1 触发条件

- 遇敌 / 主动挑衅 / 秘境陷阱 / 拍卖余波
- 由 `BlueprintCategory = combat` 路由
- 进入战斗前必须先展示战斗缘由（入战画面）

### 5.2 状态影响

- 战斗过程累积战斗日志
- 受伤 / 中毒 / 走火 / 法宝破损
- 胜利：获得战利品 + 经验
- 失败：扣气血 / 加心魔 / 被俘 / 死亡

### 5.3 可能战利品 / 修炼加成 / 减益

- 战利品：妖兽材料 / 敌人储物袋 / 法宝 / 功法 / 灵石（按 §DESIGN-VALUES.md §4 计算）
- 加成：胜利后短时间 +5% 修为（杀意余韵）
- 减益：外伤 / 内伤 / 中毒 / 心魔 +1~5

### 5.4 推荐行动（AI 参考）

- 战斗不能凭空出现，必须承接触发事件（参见 `event-blueprint.md`）
- 战斗开始后**必须默认等待玩家选择**；自运只能由玩家在当前战斗手动开启，新战斗、了结时自动关闭
- 战斗僵局（连续低伤害、无状态变化）必须触发破局时停（AI 给出诱敌、破局、脱身等应变选项）
- 战后不能自动回到叙事页；先停留结果态

### 5.5 典型叙事钩子

- 山道遇劫："留下储物袋，饶你一命"
- 秘境妖兽护巢
- 拍卖余波被人伏击
- 同门切磋点到为止

### 5.6 示例（≥3）

**示例 1（妖兽遭遇）**：入山采药，遇一头炼气五层狼妖。狼妖护崽，先发制人扑来；修士以灵剑击退，狼妖哀鸣而去。
**示例 2（邪修伏击）**：夜过山坳，被黑袍人拦路，对方自称"血煞门"，要夺储物袋。苦战半柱香，邪修逃遁，修士负伤。
**示例 3（同门切磋）**：宗门大比第一轮，与师兄斗法三百回合，平手收场。师兄笑说"三年后再战"。

## 6. 拍卖（auction）

### 6.1 触发条件

- 玩家进入坊市 / 因缘触发"拍卖因缘"
- `pendingThread.category == 'auction'` 或 `location.kind == 'auction_house'`
- 入口由事件或因果触发，不应是常驻入口

### 6.2 状态影响

- 玩家可见当前竞价者（3~5 名 BidderPersonality）
- 拍得品进入物品系统；未拍得则维持原价
- 拍卖余波（auction-aftermath）1~3 年内可能出现

### 6.3 可能战利品 / 修炼加成 / 减益

- 战利品：功法 / 法宝 / 丹药 / 材料（按拍品清单）
- 加成：拍得稀有功法 → 长期修炼速度 +X%
- 减益：高价拍下后短期灵石告急；与某竞价者结仇（pendingThread）

### 6.4 推荐行动（AI 参考）

- 拍卖场景必须完整：开场 → 介绍 → 多轮竞价 → 落锤
- 竞价者不能凭空抬价，必须按 BidderPersonality 类型出价
- 玩家输入具体出价后引擎按 DESIGN-VALUES.md §5 校验；超溢价上限必须拒绝
- 拍卖余波可承接仇家报复 / 卖家答谢 / 拍卖行任务

### 6.5 典型叙事钩子

- 卖家遮面，自称"急售"
- 竞价者中有一位认出玩家，态度骤变
- 拍品来历奇异（疑似邪修遗物）
- 拍卖行突生变故，有人搅局

### 6.6 示例（≥3）

**示例 1（功法拍卖）**：青岚城三年一度的拍卖会开场，压轴一本玄阶下品功法《潮汐引》。起拍 2720 灵石，三名散修与一名世家子弟轮番出价，最终玩家以 6800 灵石拍下。
**示例 2（法宝拍卖）**：某夜小型拍卖行拍卖一件残损法宝"残光护符"，起拍价极低。玩家拍下后，发现护符附有灵禁"残光护幕"，竟是古修遗物。
**示例 3（黑市拍卖）**：地下拍卖，卖家声称"血煞门遗产"。玩家出价后被黑袍人盯上，余波数年后以一场伏击战了结。

## 7. 秘境（secret_realm）

### 7.1 触发条件

- 玩家进入 `SecretRealmTier` 对应秘境
- `BlueprintCategory = exploration / heritage / trial`
- 入口由事件或因果触发（如：师尊指派 / 仙缘线索）

### 7.2 状态影响

- 进入秘境后获得 `environment.spirit_vein_inside` 等环境 buff
- 探索过程累积发现物品 / 触发事件
- 退出后秘境进入 `exploredRealms` 列表

### 7.3 可能战利品 / 修炼加成 / 减益

- 战利品：灵草 / 矿石 / 古修遗物 / 传承玉简
- 加成：环境灵气 +30% 修炼速度（短期）
- 减益：环境压力（-5% 防御 / 中毒概率 +20%）

### 7.4 推荐行动（AI 参考）

- 秘境必须分阶段（入口 → 浅层 → 深层 → 核心 → 退出）
- 每阶段 1~2 个事件，长度 100~200 字
- 核心区必须有传承 / 宝藏 / 战斗 三选一以上
- 退出时必须给出"带出什么"清单

### 7.5 典型叙事钩子

- 入口机关（令牌 / 口诀 / 血脉）
- 秘境核心的前代骸骨
- 灵兽守护
- 同行者遇难

### 7.6 示例（≥3）

**示例 1（采药秘境）**：青岚山东麓有一处千年灵药园遗迹，修士入内采得"九叶青"一株，带出后炼成"九叶青凝丹"，修为 +10%。
**示例 2（传承秘境）**：山崖深处发现前代剑修洞府，得玉简一枚，载残缺功法《斩潮剑诀》。同时发现骸骨旁遗下飞剑残片。
**示例 3（陷身秘境）**：误入邪修设下的"血煞困阵"，被困三日，靠同门接引方出。留下挂念"血煞门"线索，三年后以一场伏击战了结。

## 8. 因缘了结（thread_resolve）

### 8.1 触发条件

- `pendingThread.status == 'pending' && currentAge >= pendingThread.deadlineAge - 1`
- 或玩家主动了结（输入自然语言）
- 由 `BlueprintCategory = thread_resolve / emotion` 路由

### 8.2 状态影响

- 状态变 `resolved` / `failed` / `urgent`
- 视因缘类型，奖励或代价生效
- 与相关 NPC 的关系值变更

### 8.3 可能战利品 / 修炼加成 / 减益

- 战利品：依 `pendingThread.reward`
- 加成：完成因缘 → 修为 +5%~15%；声望 +10%
- 减益：失败因缘 → 心魔 +3~10；关系值跌落

### 8.4 推荐行动（AI 参考）

- 了结前必须让 AI 写出承接叙事（不能直接"完成"）
- 玩家可见史册**不能**出现"天道干预""循着旧迹与旧约继续追索"等局外词
- 因缘标题根据实际内容自然概括（如"修补血煞诀"而非"用某物余波"）
- 长文本（未了 / 已了因缘）必须能展开 / 点开查看完整内容

### 8.5 典型叙事钩子

- 三年后故人重逢
- 一句迟到的道歉
- 残页终补完功法
- 师尊遗言揭开真相

### 8.6 示例（≥3）

**示例 1（承诺了结）**：十八岁那年赴师尊之约，在青岚山之巅完成"筑基之约"。师尊点头离去，留下"再会"二字。
**示例 2（仇敌了结）**：与血煞门弟子的三年追杀因缘了结于某夜山坳，对方重伤遁走，修士负手而立。
**示例 3（血脉寻亲）**：生母谜团因缘三年到期，修士依稀线索远赴他乡，于一荒村寻得外祖母遗物，真相渐明。

## 9. 重伤（severe_injury）

### 9.1 触发条件

- 战斗失败 / 走火入魔 / 秘境陷阱 / 渡劫反噬
- `hp <= 0 || status in ['走火入魔', '雷劫反噬']`
- 由 `BlueprintCategory = combat / trial / inner_demon` 路由

### 9.2 状态影响

- `debuff.internal_injury` / `debuff.external_wound` / `debuff.poisoned`
- 寿元上限下降（重伤难返）
- 修为倒退（严重时掉小层级）
- 战斗能力暂时归零

### 9.3 可能战利品 / 修炼加成 / 减益

- 战利品：无
- 加成：痊愈后心境可能 +1（"死里逃生"印记）
- 减益：重伤期间修炼速度 ×0.30；气运 -10%

### 9.4 推荐行动（AI 参考）

- 重伤叙事必须体现"死里逃生"的紧张感
- 写入因重伤带来的心境变化（影响后续修炼 / 因缘承接）
- 不要让重伤角色立刻恢复，必须有数月 / 数年的疗伤期
- 疗伤期内可能出现因缘承接（如：有人趁火打劫 / 有人舍命相救）

### 9.5 典型叙事钩子

- 师尊闭关，闻讯破关而出
- 道侣日夜守在床前
- 仇家趁火打劫
- 自悟生死关，修为反退

### 9.6 示例（≥3）

**示例 1（战斗重伤）**：与炼气九层妖兽搏斗，险些殒命。被同门救回时，气血仅余一成，灵台受损，疗伤三载方复。
**示例 2（走火重伤）**：闭关冲击筑基失败，灵台炸裂，气血狂跌，修为倒退一层。师尊闻讯破关而出，赠丹疗伤。
**示例 3（秘境重伤）**：秘境深处触发古阵，被困七日，灵气枯竭而出，瘦骨嶙峋。外门师兄笑说"早该听人劝"。

## 10. 渡劫（tribulation）

### 10.1 触发条件

- `realm == 'great_vehicle' && cultivationExp >= expPerLevel`
- 雷劫单独走 `TribulationHandler`，不混入普通突破公式
- 由 `BlueprintCategory = trial` 路由

### 10.2 状态影响

- 渡劫中：获得 `tribulation.underway`（隐藏）
- 渡劫成功：境界升至 ascension；寿元大幅提升
- 渡劫失败：修为全废 / 身死道消 / 转世重修（视失败程度）

### 10.3 可能战利品 / 修炼加成 / 减益

- 战利品：渡劫成功 → 飞升之钥 / 天道赐福（极稀有 buff）
- 加成：成功 → 修为永久 +10%；境界特性"飞升体"
- 减益：失败 → 修为归零 / 寿元 -50% / 心魔 +50

### 10.4 推荐行动（AI 参考）

- 渡劫必须分阶段：黑云压顶 → 第一道雷 → 中间喘息 → 第七 / 第八重雷 → 第九道雷 → 飞升 / 陨落
- 玩家必须能选择"渡劫时机"（不宜过早 / 过迟）
- 渡劫画面要有强烈的天地异象感（雷云、紫电、金光）
- 渡劫前后必须承接因果（家人 / 弟子 / 仇敌的反应）

### 10.5 典型叙事钩子

- 渡劫前夕众人来送
- 雷劫中看见前世 / 已故之人
- 第九道雷迟迟不落
- 飞升瞬间天地齐喑

### 10.6 示例（≥3）

**示例 1（顺利渡劫）**：八百岁那年，紫气东来三万里，雷劫九道依次落下，修士以残光护符硬扛第八道，第九道时忽然悟通飞升之意，身化金光，破界而去。
**示例 2（渡劫失败·转世）**：第六道雷劫劈碎护体法宝，第七道直击灵台，神魂溃散。残留一缕真灵夺舍山村婴孩，记忆模糊，前缘待续。
**示例 3（渡劫失败·陨落）**：第九道雷劫超出预期，修士灵力耗尽，肉身崩解，魂飞魄散。师尊在远处遥望，沉默良久。

## 11. 飞升（ascension）

### 11.1 触发条件

- 渡劫成功 + 境界 = ascension
- 由 `BlueprintCategory = trial / emotion` 路由
- 玩家可见界面进入"终局态"（结算画面 / 新角色继承）

### 11.2 状态影响

- 境界 = ascension
- 寿元 = 99999（伪无限）
- 角色档案标记为"飞升者"
- 全部 pendingThread 标记为 `resolved`（带传承）

### 11.3 可能战利品 / 修炼加成 / 减益

- 战利品：飞升遗物 / 天道赐福（可继承给下一代）
- 加成：飞升体（修炼速度永久 +20%）
- 减益：飞升后不可回归（仅在结算时以"继承"机制续接）

### 11.4 推荐行动（AI 参考）

- 飞升叙事必须给角色一段"人间最后一眼"（回首 / 告别 / 沉默）
- 玩家可选择"继承"（新角色继承部分遗物 / 因缘）或"完结"（本角色完结）
- 飞升后**不立即进入新一轮**；先停留结算画面 1~2 拍，让玩家体会
- 不要在飞升叙事中暴露机制词（"天道""结算""存档"等）

### 11.5 典型叙事钩子

- 与道侣的最后一眼
- 把传承留给弟子
- 与仇敌一笑泯恩仇
- 飞升前最后一杯茶

### 11.6 示例（≥3）

**示例 1（完满飞升）**：修士与道侣在洞府前最后一次共饮，道侣问"还回来吗"，修士笑而不答。次日清晨，紫气东来，修士身化金光，破界而去。
**示例 2（带憾飞升）**：未了因缘"与师尊的筑基之约"早已了结，但师尊陨落百年，修士飞升时喃喃"弟子……也来了"。
**示例 3（继承飞升）**：飞升前将残光护符与《斩潮剑诀》留给弟子。弟子接令，跪送师尊离去，三十年后亦成一方剑修。

## 13. 通用 AI 接管要点

1. **承接待续**：每个事件必须承接至少 1 条 `pendingThread`，否则引擎视为"无效推进"。
2. **状态参与**：每个事件必须让 `activeStatuses` 真实参与（参见 UI-RULES Phase 2 §10/11）。
3. **去重**：同年代相同事件不重复触发（dedupe by category + age）。
4. **时间显示**：不要写"续记"；直接显示"三月后""翌日""闭关数载后"等世界内时间。
5. **同岁续写**：同年追加事件的时间标签按实际推进顺序累积显示，不能多段都卡在同一天。
6. **角色年龄一致性**：AI 写剧情前必须先判断角色当前年龄、身份、境界、身体状态、心境、近期经历、所在地点与资源；不能让 5 岁婴孩独立赴约或追查因缘。
7. **沉浸优先**：玩家可见 UI 不暴露"AI / 预加载 / 缓存 / 调试 / id"等现实词，按 `display.ts` 世界内语言展示。
