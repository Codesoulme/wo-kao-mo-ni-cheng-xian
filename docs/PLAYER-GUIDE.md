## 第一次打开游戏

1. 打开浏览器，访问项目首页（开发模式下通常是 `http://localhost:5176` 或 `http://localhost:3000`）。
2. 页面中央会出现 **开始** 按钮。点它。
3. 系统会让你：
   - **配置 AI 端点**（必填；详见下一节）
   - **创建角色**（名字、灵根、出身、天赋）

如果你跳过 AI 配置直接创建角色，系统会用世界内的口吻提示你"天机未启"，然后拒绝继续推进。

---

## 必须先做的两件事

### 1. 配置 AI 端点

点击右上角 **配置**（齿轮图标），填入：

| 字段 | 说明 | 示例 |
|------|------|------|
| API Base URL | OpenAI 兼容协议的接入地址 | `https://ark.cn-beijing.volces.com/api/v3` |
| API Key | 你的端点密钥 | `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |
| 模型 ID | 你要使用的模型标识 | `doubao-pro-32k` |

保存后点 **测试连接** 验证。验证通过才能推进游戏。

> **为什么不内置**：本项目不绑定任何 AI 服务商，玩家可以自由选择（火山方舟、OpenAI、Anthropic 代理、自部署端点等），按使用量付费给服务商。配置仅保存在你本地，不上传仓库。

### 2. 创建角色

填写或随机生成：

- **名字**：你在修仙世界里的道号
- **灵根**：决定修炼速度与功法适配（金 / 木 / 水 / 火 / 土 / 异灵根）
- **出身**：决定初始资源、关系网与开局事件（散修 / 世家 / 宗门弟子 / 凡人转生 等）
- **天赋**：先天特长，可能影响后续事件走向

填好后点 **入道**，游戏正式开始。

## 界面上你看到的每一块

主界面大致分这几块（从上到下、从左到右）：

### 顶部状态栏（永远可见）

- 境界（金丹 / 元婴 / …） + 修为进度
- 年龄 / 当前年号
- 心魔值（越高越危险）
- 3 个最关键的 **状态标签**（点击展开详情）
- 2 个 **体质标签**（独立显示）
- 因缘数量（红色小点表示有紧迫因缘）

### 角色详情卡（侧拉）

- 完整属性（精 / 气 / 神 / 体 / 魄 / 寿元 / 灵根契合度）
- 已习功法 / 已习法术与灵禁
- 装备
- 当前所有状态（含折叠展开）

### 主叙事区（中央）

- 本年度的剧情（默认流式生成，逐段出现）
- 关键事件、选择、NPC 对话都在这里

### 因缘卡（叙事下方）

- 你还没还的债、还没报的仇、还没赴的约
- 紧迫因缘（带 deadline）排在最前面
- 点击展开看完整描述与可能的行动

### 库存 / 坊市 / 阵法 / 灵宠

- 点底部对应 Tab 打开
- **库存** 默认只显示最近获得 + 已装备；点 **查看全部** 展开
- **坊市** 同年货架不刷新（避免被刷钱）；切换年才会重置
- **灵宠** 按亲密度排序；可喂养 / 训练 / 解契

### 战斗 / 突破 / 渡劫 / 飞升 弹层

- 都是 **模态弹层**，不会自己跳出来
- 你可以随时关闭、暂存当前状态

## 一次推进会发生什么

点 **推进** 按钮后：

1. 引擎读取你当前的状态快照（年龄、境界、心魔、库存、装备、因缘）。
2. 引擎把这些信息 + 近期历史 + 因缘状态 打包成 Prompt，发给 AI。
3. AI 生成：本年度的剧情、可能的事件、给到你的选择、是否触发战斗或因缘。
4. 引擎 **校验** AI 的输出：是否有越级、是否有不合规物品、是否与已知世界矛盾。
5. 通过校验的内容写入你的存档；失败的被引擎丢弃并由 AI 重新生成（最多 3 次）。
6. 前端刷新顶部状态栏 / 因缘卡 / 主叙事区。

整个过程通常 5–15 秒。期间你会看到 **悬浮的"天道权衡中…"** 提示——这是世界内表达，不会显示"AI 加载中"。

> 如果推进长时间无响应，先检查右上角 **配置** 里的 AI 端点是否可用。

## 战斗怎么打

战斗是 **回合制**，进入战斗后默认 **等待你出手**，不会自动打。

### 出招顺序

每回合由 **速度** 决定先后手。每个角色每回合可以：

1. 攻击
2. 使用技能（基于已习法术 / 灵禁 / 法宝自带术式）
3. 使用物品（丹药、符箓、阵旗）
4. 防御 / 蓄势
5. 尝试脱逃（成功率与境界差 / 心魔值相关）

### 自动战斗

点 **自运** 开启自动战斗后，系统会按简单策略替你出招。

> 重要规则：**自运必须由玩家在当前战斗中明确手动开启**，新战斗 / 开战缘由页 / 战斗结束时会 **自动关闭自运**。这是为了防止你忘了关而被坑。

### 战利品

战胜后可能掉落：

- 物品（灵石 / 丹药 / 法宝 / 材料）
- 修为（计入修为进度）
- 因缘（NPC 关系变化、获得新因缘）

战利品命名规则：**只保留器物本名**，不会出现"某某的遗下 XX""某某的护符"。归属与来历写在物品描述里。

## 突破怎么过

突破不是按一下按钮就完事的，它分 **4 个阶段**：

1. **准备**（数月）：你需要闭关、服用辅助丹药、调整心境、积累状态。
2. **触发**：当你满足条件，可点 **尝试突破**。
3. **试炼**：可能触发天劫、心魔、问心、功法反噬、战斗中突破等。
4. **转化**：成功则提升境界并获得增益；失败则可能掉阶 / 重伤 / 获得负面状态。

突破分 5 档品质：

| 品质 | 效果 |
|------|------|
| 普通 | 仅提升境界 |
| 良品 | 提升境界 + 少量属性加成 |
| 上品 | 提升境界 + 多属性加成 + 小幅天赋提升 |
| 极品 | 提升境界 + 全面属性加成 + 解锁天赋分支 |
| 神品 | 提升境界 + 全部属性加成 + 触发隐藏事件 |

> 准备越充分、心境越平和、辅助越对路，品质越高；强行突破多半是普通甚至失败。

## 因缘（未了事项）怎么看

因缘 = 你的"未了事项"，包括：

- 约定（你答应 NPC 某时某刻要做的事）
- 仇恨（NPC 与你有未解的恩怨）
- 恩情（NPC 帮过你，你还欠一份情）
- 机缘（你路过某个地方埋下的线索）
- 旧账（你欠宗门的、欠家族的）

**怎么看优先级**

- 顶部状态栏的红色数字 = 紧迫因缘数量
- 紧迫因缘会带 deadline（具体到某年某月）
- **不还** 不会立刻死，但会持续积累心魔，最终可能在渡劫或化神时爆发

**怎么还**

- 主动去因缘卡里点 **查看详情**，会看到具体地点、NPC、可选行动
- 有些因缘会自动在你推进时触发（比如路过当年埋下机缘的地方）

## 常见误区

### ❌ "我直接输入'让我飞升'试试"

不会成功。AI 会按修仙世界内话术自然拒绝——比如"道友此时连筑基都未稳，谈何飞升"。**这是引擎硬约束的结果**，不是 bug。

### ❌ "我把所有丹药一次性吃掉"

部分丹药有副作用（走火入魔、丹毒累积）。部分丹药同类不能叠加。引擎会自动校验，违规的服用会被引擎丢弃并提示。

### ❌ "我去坊市刷钱"

同年坊市的货架 **不会因为买卖、切页、重进而刷新**，只有跨年才会重置。刷钱路线不通。

### ❌ "我让小孩独立去查真相"

不行。0-6 岁角色只会被写成在大人的看护下玩耍、听闻传说、被家人带去。**不会** 写成独立赴约、追查因缘。

### ❌ "我强行让幼年角色同岁承接大量线索"

不会发生。引擎对幼年角色的承接会改写成"暂缓"、"被带去"、"懵懂盼头"等合理承接。

## 进阶玩法

### 多开档对比策略

每个档是完全独立的种子。可以开多个档对比：

- 不同灵根 → 不同突破节奏
- 不同出身 → 不同开局关系网
- 不同自定义介入（interfere）→ 不同 AI 反应

### 介入（interfere）

游戏中随时可以点 **介入**，输入你想做但游戏没主动给你的事。比如：

- "我想今夜去探那个山洞"
- "我想和这位师叔单独谈谈"

系统会按修仙世界内话术回应，**合理的事会被采纳**，**不合理的事会被自然拒绝并推动替代剧情**。

> **不要** 用介入来要 buff、要物品、要越级——这些都会被边界校验拦截。

### 归档与读档

点 **归档**（右上角菜单）可保存当前世界到一个槽位。最多 5 个归档槽位。读档从归档列表点 **恢复** 即可。

### 重置世界

点 **重置世界**（菜单底部，红色按钮）会 **用游戏内确认弹窗**（不是浏览器原生 confirm）让你二次确认。重置后回到创角界面，所有本地数据清除。
